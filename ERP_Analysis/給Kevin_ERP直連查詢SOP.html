二、連線基礎模板

建議直接複製成 erp_query.py：

"""ERP 直連查詢 + 匯出 Excel 範本"""
import os
import pyodbc
import pandas as pd
from dotenv import load_dotenv

load_dotenv()  # 載入 .env

# ── 連線字串 ──
conn_str = (
    f"DRIVER={{{os.getenv('ERP_ODBC_DRIVER', 'ODBC Driver 17 for SQL Server')}}};"
    f"SERVER={os.getenv('ERP_SERVER')},{os.getenv('ERP_PORT')};"
    f"DATABASE={os.getenv('ERP_DATABASE')};"
    f"UID={os.getenv('ERP_USER')};"
    f"PWD={os.getenv('ERP_PASSWORD')};"
    "TrustServerCertificate=yes;"
    "timeout=10;"
)


def run_query(sql: str, params: tuple = ()) -> pd.DataFrame:
    """執行 SQL，回傳 DataFrame"""
    conn = pyodbc.connect(conn_str)
    try:
        df = pd.read_sql(sql, conn, params=params)
        # 清 SQL Server CHAR 欄尾端空白
        for col in df.select_dtypes(include='object').columns:
            df[col] = df[col].str.rstrip() if hasattr(df[col], 'str') else df[col]
        return df
    finally:
        conn.close()


def export_to_excel(df: pd.DataFrame, filename: str):
    """匯出 Excel"""
    os.makedirs("output", exist_ok=True)
    path = f"output/{filename}"
    df.to_excel(path, index=False)
    print(f"✅ 已匯出：{path}（{len(df)} 筆）")


# ── 範例 1：採購單（近 30 天）──
if __name__ == "__main__":
    sql = """
    SELECT TOP 100 *
    FROM v_ht_purchase_order
    WHERE order_date >= DATEADD(day, -30, GETDATE())
    ORDER BY order_date DESC
    """
    df = run_query(sql)
    print(df.head())
    export_to_excel(df, "採購單_近30天.xlsx")

執行：

python erp_query.py

四、常用查詢範例

1. 某客戶的所有訂單（近 90 天）

SELECT *
FROM v_ht_customer_order
WHERE customer_name LIKE N'%晶達%'
  AND order_date >= DATEADD(day, -90, GETDATE())
ORDER BY order_date DESC

2. 某品號的所有銷售明細

SELECT *
FROM v_ht_customer_order_lines
WHERE part_no = 'JD14A-5Z0'
ORDER BY order_date DESC

3. 採購金額統計（依廠商，近一年）

SELECT
    vendor_name,
    COUNT(*) AS 採購單數,
    SUM(total_amount) AS 採購總金額
FROM v_ht_purchase_order
WHERE order_date >= DATEADD(year, -1, GETDATE())
GROUP BY vendor_name
ORDER BY 採購總金額 DESC

4. 製令單途程（含線別廠商）

SELECT *
FROM v_ht_mo_routing
WHERE order_no = '25040001'
ORDER BY process_seq

六、疑難排解

連不上 ERP？

  • 檢查：是否在公司網路內（192.168.0.201 是內網 IP）
  • 檢查：ODBC Driver 裝了沒（見第一章第 1 點）
  • 改連線字串裡 timeout=30 延長

查出來的中文是亂碼？

  • SQL Server 用的是 BIG5，確認 pd.read_sql 讀出來是 str 型
  • 若真的亂碼：df = df.applymap(lambda x: x.encode('latin1').decode('big5', errors='ignore') if isinstance(x, str) else x)

查詢跑很慢？

  • 一定要加 WHERE 條件限縮日期範圍
  • 用 TOP N 先測試
  • 避免 SELECT * 撈全部欄位
  • 超大查詢用 pandas 的 chunksize 分批讀

SQL 錯誤？

把錯誤訊息連同 SQL 丟給 Claude，請他除錯。

七、不要做的事

❌ 絕對不要原因
INSERT / UPDATE / DELETE會汙染正式 ERP 資料
DROP TABLE / TRUNCATE災難
sp_executesql 亂跑有注入風險
把 ERP 資料 push 上 GitHub機敏資料外洩
把 .env 傳給別人帳密外洩
長時間鎖表的大查詢（百萬 Join）影響 ERP 前端操作
